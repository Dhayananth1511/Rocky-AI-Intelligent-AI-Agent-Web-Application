# 🤖 Rocky AI – Intelligent AI Agent Web Application

Rocky AI is a **ChatGPT‑style AI assistant web application(Only Chat)** built using Python and Flask.  
It demonstrates how to build and deploy a real **AI agent architecture** using a cloud‑hosted language model, with a modern UI and scalable backend.

---

## 🚀 Features

- 💬 Conversational AI chatbot (ChatGPT‑like)
- 🧠 AI model integration via Hugging Face API
- 🗂️ Modular agent‑based architecture
- 🧾 Conversation memory handling
- 🔐 Secure API key management using environment variables
- 📱 Fully responsive UI (mobile + desktop)
- 🌐 Cloud deployment with public access

---

## 🧠 AI Model Information

- **Model Name:** `meta-llama/Meta-Llama-3-8B-Instruct`
- **Model Provider:** :Meta
- **API Platform:** :Hugging Face Router API

⚠️ *Note:*  
This model does **not provide real‑time news or live internet access** by default.  
Live data requires external APIs.
